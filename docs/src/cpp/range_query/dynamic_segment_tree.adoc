Range Sum Query

動的セグメント木(Dynamic Segment Tree)

== 概要

セグメント木の動的バージョン

大きい区間(区間$[0, 10^9)$等)を扱えたり、メモリ量少なめに区間クエリを処理することができる。

== 計算量

扱う区間のサイズをNとする。(つまり、区間$[0, N)$を扱う動的セグメント木)

- 更新: $O(\log N)$
- 区間和: $O(\log N)$
