= kd-tree (2次元)

kd-tree (k-dimensional tree)

== 概要

2次元平面上の頂点を木構造で管理する。

== 計算量

- 構築: $O(N \log^2 N)$
- 領域探索クエリ: $O(\sqrt{N} + K)$ ($K$は出力する点の数)
- (一様分布の点集合における)最近傍点クエリ: $O(\log N)$
