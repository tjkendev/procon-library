Radix Heapによるダイクストラ法 (Dijkstra's Algorithm with Radix Heap)

== 概要

単一始点最短経路問題(SSSP)を解くアルゴリズム。到達するまでのコストが小さい方からコストを伝搬させていく。

Radix Heapを用いた実装。負辺が含まれないグラフ上で利用可能。

== 計算量

コストの最大が$D$とすると $O((|E| + |V|) \log D)$
