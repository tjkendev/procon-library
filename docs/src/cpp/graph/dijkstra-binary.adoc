= 単一始点最短経路 (ダイクストラ法, 二分ヒープ)
:title: {doctitle} - {pagetitle}

二分ヒープによるダイクストラ法 (Dijkstra's Algorithm with Binary Heap)

== 概要

単一始点最短経路問題(SSSP)を解くアルゴリズム。
到達するまでのコストが小さい方からコストを伝搬させていく。

負閉路を含まないグラフに対して適用できる。

== 計算量

$O((|E| + |V|) \log |V|)$
