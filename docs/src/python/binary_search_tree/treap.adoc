== 概要

乱択アルゴリズムを用いた平衡二分探索木。
平衡二分探索木の中で実装が簡単なアルゴリズム。

木の高さは$O(\log N)$になる。

== 計算量

各クエリ $O(\log N)$

== 実装

各ノードに以下の情報を持たせたバージョン

- 左右の子ノード
- key
- priority
- そのノード以下の部分木のサイズ

高速に処理するため、非再帰でノードの追加、削除を行うようにしている。
