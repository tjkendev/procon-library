= 素数mod上の平方根 (Tonelli-Shanks algorithm)
:title: {doctitle} - {pagetitle}

square roots modulo a prime

== 概要

ある奇素数 $p$ と $p$の平方剰余(Quadratic residue) となる $y$ について $x^2 \equiv y$ (mod $p$) となる $x$ を求めるアルゴリズム。
