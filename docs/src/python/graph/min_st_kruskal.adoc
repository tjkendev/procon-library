= 最小全域木 (クラスカル法)

クラスカル法 (Kruskal's Algorithm)

== 概要

グラフ latexmath:[G] 上の最小全域木(Minimum Spanning Tree)を求める。

クラスカル法では、コストが小さい辺から、連結でない頂点同士を繋いでいくことで、最小全域木を求める。

== 計算量

latexmath:[O(|E| \log |V|)]
