= dsu on tree

== 概要

根付き木において、各頂点に対し、その頂点を根とする部分木に関する特性を求めるためのテクニック

== 計算量

木の頂点数を latexmath:[N] とする。

特性を計算するためのデータ構造への1頂点の追加に latexmath:[O(A)], 1頂点の削除に latexmath:[O(B)] 、1頂点の取得に latexmath:[O(C)] かかる場合

latexmath:[O(N (A + B) \log N + NC)]
