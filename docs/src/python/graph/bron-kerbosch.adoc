最大クリーク問題 (maximum clique problem)

== 概要

グラフ上の最大クリークを求める。

Bron-Kerbosch Algorithm はバックトラックでグラフ上の極大クリーク(maximal clique)を全て列挙する。

== 計算量

頂点数を $N$ とする時、 $O(3^{N/3})$

特に疎グラフに関しては $O(Nd 3^{d/3})$ ($d$ は グラフの degeneracy)
