D'Esopo-Pape Algorithm

== 概要

単一始点最短経路問題(SSSP)を解くアルゴリズム。dequeを用いて計算する。

負辺に対する計算はできるが、負閉路が含まれない必要がある。

大体のケースにおいて、Dijkstra法やBellman-Ford法よりも高速になるらしいが、最悪ケースは指数オーダーになる。
