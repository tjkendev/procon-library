幅優先探索 (Breadth First Search)

== 概要
単一始点最短経路問題(SSSP)を解くアルゴリズム。

グラフ$G = (V, E)$において、辺のコストが全て1である場合、BFSで探索することで、1頂点から各頂点までの距離を線形オーダーで探索できる。

== 計算量

$O(|V| + |E|)$
