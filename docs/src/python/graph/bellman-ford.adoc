ベルマンフォード法 (Bellman-Ford Algorithm)

== 概要

重み付きグラフ$G = (V, E)$の始点$v_s$から終点$v_t$への最短経路を求める。

辺のコストに負が含まれていても正しく動き、負閉路も検出することができる。

== 計算量

$O(|V||E|)$
