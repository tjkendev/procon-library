= 橋検出 (Path-based DFS)

橋検出(Bridge-finding)

== 概要

グラフ$G = (V, E)$において橋(除去するとグラフが連結でなくなる辺)を検出する。

== 計算量

$O(|V| + |E|)$

== 実装

橋検出して縮約したグラフを返す実装。
多重辺にも対応できるようにしている。
