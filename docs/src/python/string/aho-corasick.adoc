= Aho-Corasick Algorithm
:title: {doctitle} - {pagetitle}

== 概要

文字列探索アルゴリズム。

ある文字列$T$について、$N$個の文字列$s_1, s_2, ..., s_N$の中からマッチする(最長の)文字列を線形に探索することができる。

== 計算量

$O(|T| + \sum_i |s_i|)$
