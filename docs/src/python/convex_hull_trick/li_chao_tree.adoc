== 概要

Li-Chao (Segment) Treeはセグメント木を用いてConvex Hull Trickを行うアルゴリズム。

詳細はこちら(ブログ): link:http://smijake3.hatenablog.com/entry/2018/06/16/144548[Li Chao Treeのメモ]

== 計算量

- 直線追加: $O(\log N)$
- 線分追加: $O(\log^2 N)$
- 最小値計算: $O(\log N)$
