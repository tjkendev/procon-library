= 最近点対 (分割統治法)

== 概要

2次元平面の latexmath:[N] 個の点が与えられた時、最近点対(最も近い頂点のペア)の距離を計算する。

分割統治法で、ソートされた頂点を分割し、最小距離となりうる頂点ペアの距離を計算しながらマージしていく。

== 計算量

latexmath:[O(N \log N)]
