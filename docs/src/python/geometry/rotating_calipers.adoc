= 最遠点対 (キャリパー法)
:title: {doctitle} - {pagetitle}


== 概要

2次元平面の$N$個の点が与えられた時、最遠点対(最も遠い頂点のペア)の距離を計算する。

キャリパー法 (Rotating Calipers)では、一度凸包を求め、求めた凸多角形を回転しながら走査し、最遠点となる頂点ペアを求める。

計算量は$O(N \log N)$
