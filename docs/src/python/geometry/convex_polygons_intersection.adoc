= 凸多角形同士の交差判定/交点

Intersection of Convex Polygons

== 概要

凸多角形同士の交差判定を行う。

2つの凸多角形 $P = (v_1, v_2, ..., v_N)$, $Q = (w_1, w_2, ..., w_M)$ の交差判定を行う。

それぞれの凸多角形のある辺から始め、外積によって辺の位置関係を判定しながら走査し交差している箇所を求める。

== 計算量

$O(N + M)$
