線分同士の交差判定

概要

線分\(P_0-P_1\)と線分\(Q_0-Q_1\)が交差しているかを\(O(1)\)で判定する。 外積と内積を用いて以下のように判定できる。

  1. 外積を用いて、以下の2つの外積の積を計算

    • \(O_0 = |(P_1 - P_0) \times (Q_0 - P_0)| |(P_1 - P_0) \times (Q_1 - P_0)|\)

    • \(O_1 = |(Q_1 - Q_0) \times (P_0 - Q_0)| |(Q_1 - Q_0) \times (P_1 - Q_0)|\)

  2. \(O_0 \le 0\) かつ \(O_1 \le 0\) であり、 \(O_0 < 0\) もしくは \(O_1 < 0\) である場合は交差

  3. \(O_0 = O_1 = 0\)である場合は \(P_0 = (1, 0), P_1 = (2, 0), Q_0 = (3, 0), Q_1 = (4, 0)\)のように線分が一直線上に並んでいるため、さらに内積を用いて判定

    • \(P_0\)と\(P_1\)の距離\(A\) と 内積 \(I_0 = (P_1 - P_0) \cdot (Q_0 - P_0)\), \(I_1 = (P_1 - P_0) \cdot (Q_1 - P_0)\)を計算

    • 区間\([0, A^2\)]と区間\([I_0, I_1\)]が重なるかを判定 (重なっていれば交差)

実装

AOJ: "CGL_2_B: Segments/Lines - Intersection"で提出した実装

# 線分同士の交点判定
def dot3(O, A, B):
    ox, oy = O; ax, ay = A; bx, by = B
    return (ax - ox) * (bx - ox) + (ay - oy) * (by - oy)
def cross3(O, A, B):
    ox, oy = O; ax, ay = A; bx, by = B
    return (ax - ox) * (by - oy) - (bx - ox) * (ay - oy)
def dist2(A, B):
    ax, ay = A; bx, by = B
    return (ax - bx) ** 2 + (ay - by) ** 2
def is_intersection(P0, P1, Q0, Q1):
    C0 = cross3(P0, P1, Q0)
    C1 = cross3(P0, P1, Q1)
    D0 = cross3(Q0, Q1, P0)
    D1 = cross3(Q0, Q1, P1)
    if C0 == C1 == 0:
        E0 = dot3(P0, P1, Q0)
        E1 = dot3(P0, P1, Q1)
        if not E0 < E1:
            E0, E1 = E1, E0
        return E0 <= dist2(P0, P1) and 0 <= E1
    return C0 * C1 <= 0 and D0 * D1 <= 0
for q in range(int(input())):
    x0, y0, x1, y1, x2, y2, x3, y3 = map(int, input().split())
    print(+is_intersection((x0, y0), (x1, y1), (x2, y2), (x3, y3)))
