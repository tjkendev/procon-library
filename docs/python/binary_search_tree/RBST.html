Randomized Binary Search Tree (RBST)

概要

乱択アルゴリズムを用いた平衡二分探索木。

サイズ\(p\)の木Aとサイズ\(q\)の木Bをマージする際、確率\(\frac{p}{p+q}\)で木Aの根、確率\(\frac{q}{p+q}\)で木Bの根をマージ後の根頂点にする、を再帰的に繰り返し、マージ後の木を構築。 木の高さを\(O(\log N)\)にする。

計算量

各クエリ \(O(\log N)\)

実装

merge-splitベース

各ノードに以下の情報を持たせている

  • 左右の子ノード

  • key

  • そのノード以下の部分木のサイズ

高速化のために非再帰で処理するようにしている。

# split a tree using stack info
def __split(st, dr):
    l = None; r = None
    st.reverse(); dr.reverse()
    for x, d in zip(st, dr):
        if d:
            x[1] = q = l
            p = x[0]
            l = x
        else:
            x[0] = p = r
            q = x[1]
            r = x
        x[3] = (p[3] if p else 0) + (q[3] if q else 0) + 1
    return l, r

# split tree t into two trees of size k and |t|-k
def split(t, k):
    if not t:
        return None, None
    x = t
    st = []; dr = []
    while x:
        l = x[0]
        c = (l[3] if l else 0) + 1

        st.append(x)
        if c <= k:
            k -= c
            x = x[1]
            d = 1
        else:
            x = x[0]
            d = 0
        dr.append(d)
    return __split(st, dr)
