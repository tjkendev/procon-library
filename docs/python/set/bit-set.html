ビット集合演算

概要

集合\(S\)を1つの数値のbitで扱う。 具体的に、要素\(i\)が含まれることを S & (1 << i) > 0 で表す。

部分集合の列挙

\(T \subseteq S\) となる部分集合\(T\)を列挙する。

# 集合Mの部分集合を列挙
# R が結果
R = []
v = (-1) & M
while v:
    R.append(v)
    v = (v - 1) & M

サイズ\(K\)の部分集合の列挙

蟻本ベース[1]の実装

\(T \subseteq \{0, 1, ..., N-1\}\), \(|T| = K\)となる部分集合\(T\)を列挙する。

# サイズKの部分集合を列挙
# R が結果
R = []
v = (1 << K) - 1
while v < (1 << N):
    R.append(v)
    x = v & -v; y = v + x
    v = ((v & ~y) // x >> 1) | y
