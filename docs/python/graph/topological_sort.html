トポロジカルソート(Topological sort)

概要

トポロジカルソートは、DAGなグラフ\(G = (V, E)\)のトポロジカル順序を求める。

Kahn’s Algorithmでの実装。

計算量

\(O(|V| + |E|)\)

実装
