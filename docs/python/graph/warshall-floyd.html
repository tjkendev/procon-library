全点対最短経路(ワーシャルフロイド法)

ワーシャルフロイド法 (Warshall-Floyd Algorithm)

概要

グラフ\(G = (V, E)\)の全てのペア\((v, w)\)間の最短経路コストを求める。

計算量は \(O(|V|^3)\)

実装

# cost[i][j]: 頂点v_iから頂点v_jへ到達するための辺コストの和
for k in range(V):
    for i in range(V):
        for j in range(V):
            if cost[i][k]!=INF and cost[k][j]!=INF:
                cost[i][j] = min(cost[i][j], cost[i][k] + cost[k][j])
